# SmartSpend – Personal Finance Tracker

A Python desktop application that analyzes credit card transaction data 
to help users understand their spending habits and manage their budget.

## Phase 1 – Data Pipeline and EDA
This phase covers the initial data processing pipeline including:
- Loading and cleaning the transaction dataset
- Handling missing values using mean and mode imputation
- Removing duplicate and invalid records
- Feature engineering (Month, Year, DayOfWeek, IsWeekend)
- Exploratory data analysis and financial metrics
- Generating 6 visualizations saved to the charts/ folder
- Writing a plain text summary report
